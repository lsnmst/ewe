# Cosmoscape of the Yoruba plants

An interactive web-based visualization of Yoruba plant knowledge, formulae, and cosmological spatial relationships.
Built with Svelte, Leaflet, and a custom tile-based geospatial rendering of the Yoruba cosmoscape.

This project allows users to:

- Explore plants (Ewé) and their cosmological positions
- Browse Ifá formulae and the plants associated with each
- View plant details, ritual functions, and IUCN conservation status
- Visualize spatial clustering through a symbolic map
